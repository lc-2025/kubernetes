# Log Output

Prints a random hash every 5 seconds.

## Getting started

On terminal, from project root:

- To run the script

```bash
  npm run start
```

### Using Docker

You may use Docker to build the container and run the image required in the deployment step.

On terminal, from project root:

Build the image and run the container

```bash
docker-compose up
```

Stop the container

```bash
docker-compose stop
```
